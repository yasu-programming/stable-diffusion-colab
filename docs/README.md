## ⚖️ ライセンスについて

本プロジェクトで推奨するモデルは全て**商用利用可能**ですが、生成時は以下に注意してください：
- 実在人物の肖像権
- 著作権のあるキャラクター・ロゴの模倣
- 有害コンテンツの生成禁止
